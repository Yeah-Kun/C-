# 项目
## RocketGroundStation
### 简介
精确制导灭火弹项目地面站，地面识别并锁定物体，发射火箭，火箭持续根据识别框进行姿态调整，最终命中目标。

### 开发环境
- Win10 系统
- Qt 5.11
- OpenCV 3.4.2 + contrib

### 功能
- [x] 保险开关
- [x] 框住目标并锁定目标
- [x] 持续跟踪目标
- [ ] 发射火箭
- [ ] 根据回传图像进行姿态调整
- [ ] 紧急制动（急停）

### 其他
- 识别框输出格式（Rect.txt）
```
std::fstream outFile;
outFile << center_x << "," << center_t << "," << track_x << "," << track_y << "," << width << "," << height;
```
